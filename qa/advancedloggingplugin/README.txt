==========
advancedlogging
==========

About
------------------

Licensed under the Apache Software License, 2.0

You can install it with "python setup.py install"

What It Does
------------

nose-advancedlogging is a plugin to the nose test framework which provides a
faculty for receiving user friendly html logs.

Command line options
--------------------

After it is installed, the plugin adds the following command line flags to 
nosetests::

    --advanced-logging  Enables advanced logging.

    --disable-capture-screen  Optional: Disables screen capturing on failure.

    --advanced-log-filename  Option: name of html log file that will be created.
